# hackerthon1-ministory

## 서비스 간단 소개
### MINISTORY
나만의 게시글을 올릴 수있는 게시글 사이트 만들기

### 기능 소개
- 나만의 페이지에, 나만의 카테고리를 만들어, 나만의 게시글을 작성할 수 있다. 
- 댓글, 좋아요, 스크랩을 할 수 있다.

### 기술 스택
- frontend
  - thymeleaf
  - jquery

- backend
  - Java 11
  - Spring Boot
  - Spring Data JPA

- DB
  - MySQL

### 진행 상황
- 게시물
  - 20%
  - 구현 중
- 댓글
  - 백엔드 완료
  - 프론트 구현해야함
- 카테고리
  - 구현 중
- 좋아요, 스크랩
  - 백엔드 50%
  - 프론트 구현해야함
- 로그인
